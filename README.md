# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview

Heart Disease Prediction is a **Machine Learning web application** that predicts whether a person is at risk of heart disease based on medical parameters.

The model is trained using medical dataset features such as age, cholesterol level, blood pressure, and other health indicators.

The web interface is built using **Flask** and deployed as a simple prediction system.

---

## 🚀 Features

* Predict heart disease risk using ML
* Simple and user-friendly web interface
* Real-time prediction
* Lightweight Flask web app
* Easy deployment on cloud platforms

---

## 🧠 Machine Learning Model

The model is trained using classification algorithms to determine whether a patient has heart disease.

### Input Features

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol Level
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise Induced Angina
* ST Depression
* Slope
* Number of Major Vessels
* Thalassemia

## ⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/heart-disease-ml.git

Move to project folder:

cd heart-disease-ml

Install required libraries:

pip install -r requirements.txt

Run the Flask app:

python app.py

---

## 🌐 Usage

1. Open the web application.
2. Enter patient medical details.
3. Click **Predict**.
4. The system will show whether heart disease is detected or not.
